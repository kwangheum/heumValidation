event

Event return Description
reset() undefined 선택된 validation 안의 내용을 초기화 시킵니다
parseBoolean(child element) boolean parseBoolean
emailValidation(child element) boolean 해당 요소가 이메일이 유효한지 검사합니다
phoneValidation(child element) boolean 해당 요소의 연락처가 유효한지 검사합니다
isEmpty(child element) boolean 해당 요소가 빈값인지 체크합니다
validations(child element) undefined 해당 요소를 검사합니다
changeView(element,division,text) undefined (해당 기능은 jquery.bootstrap.heum.validation.js에만 해당됩니다)
division에 따라 요소 색이 변경됩니다
division 종류는 error,warning,success가있습니다
text는 원하는 validation 글자를 표시합니다

option checkbox나 radio는 같은 name의 첫번째 요소의 옵션만 인식합니다

Data Type Default Description
data-heum-min Integer 0 최소로 입력해야할 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-heum-max Integer 0 최대 입력할 수 있는 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-heum-label String null 해당 요소의 라벨 이름
data-heum-group-label String null radio 또는 checkbox일때 묶일 그룹의 라벨 이름
data-heum-null boolean true 빈칸 허용 할지의 유무
data-heum-number boolean false 숫자만 사용할지의 유무
data-heum-validation boolean true 해당 요소를 검사할지 말지의 유무
